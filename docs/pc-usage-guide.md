# AI抖商学院 PC 客户端 · 使用说明（大白话版）

> 这份文档是写给 **使用者 / 测试 / 客服** 的，不讲源码。
> 想看打包流程或开发细节，看 `apps/web/PACKAGING.md`。
> 本文对应的 PC 端版本：v1.0.6 起（窗口可自由缩放 + 多端头像同步两个修复合入后）。

---

## 一、客户端是什么形态

把这个 PC 客户端想成 **「一个软件进程，可以同时开很多扇窗子，每扇窗子里登一个账号」**。

- 同一个账号，可以同时开好几扇窗（左边盯群、右边盯私聊都行）。
- 不同账号，可以同时挂在同一台电脑上，互不干扰、不用来回切。
- 软件全部关掉再打开，**还会自动恢复你上次登过的账号**，不用重新输密码。
- 软件最小化到系统托盘，**点托盘图标就能呼出**，不会真的退出。

---

## 二、五种"开窗"场景

下面五种情况都会蹦出一扇窗，本质都是同一种"标准窗"，**都能自由缩放、最大化、全屏**（最小尺寸 360×480，跟手机一样窄都行）。

### ① 主窗口（第一次打开软件）

桌面双击图标，第一次打开，软件会自动给你建一个本地档案叫"账号 1"，弹一扇登录窗让你登。
👉 **第一次开软件那扇窗。**

### ② 同账号多开（`Ctrl+N` / macOS: `Command+N`）

已经登录了，但想同时摆两扇窗——比如左边盯客户群、右边盯老板私聊。
按快捷键，或菜单 **窗口 → 新建窗口（同账号）**，立刻再开一扇属于当前账号的新窗。

- 两扇窗共享同一个登录态、同一份消息记录。
- 不会重新登录、不会断线。
- 想关哪扇关哪扇，互不影响。

👉 **同一个账号，多开几扇窗一起看。**

### ③ 不同账号多开（`Ctrl+Shift+N` / macOS: `Shift+Command+N`）

已经登了"账号 1"（工作号），还想同时挂"账号 2"（小号 / 客服号），不想退出来回切。
按快捷键，或菜单 **窗口 → 新建窗口（另一个账号）**，会建一个全新的本地档案，弹登录页让你登第二个账号。

- 两个账号的消息、好友、设置完全隔离。
- 关机重启后，所有挂着的账号都还在。

👉 **电脑上同时挂俩号、仨号，每个号一扇窗，互不影响。**

### ④ 账号管理里点已有账号

之前登过几个号都关掉了，下次想直接打开其中某一个，不用重新输密码。
菜单 **窗口 → 账号管理**，列出所有登过的账号，点哪个就拉起哪个的窗，自动恢复登录态。
也可以 **右键托盘图标** → 选要打开的账号。

👉 **软件记得你登过哪些号，下次想开哪个就点哪个，不用重新登。**

### ⑤ 二次双击图标（已经在后台跑着）

软件已经最小化在托盘里跑着了，你忘了，又在桌面双击图标。

- 软件 **不会被启动两次**（有"单例锁"）。
- 第二次启动会立刻退出，并通知已经在跑的进程："用户又点我了，再开一扇窗给他"。
- 已运行的进程会用 **最近用过的那个账号** 顺手再补开一扇窗。

👉 **后台跑着的时候你又点了下图标，它不会重启自己，会再给你开一扇窗（用最近在用的号）。**

## 三、窗口可以自由调节大小

每一扇窗（不管是上面五种里的哪种）都支持：

- 拖窗口四个角 / 四条边 → **缩放**（最小可拖到 360 × 480，跟手机宽度差不多）。
- 双击标题栏 / 点最大化按钮 → **最大化 / 还原**。
- 菜单 **显示 → 切换全屏** 或快捷键 → **进入 / 退出全屏**。
- 拖标题栏 → **任意移动**。

> 旧版本（< v1.0.6）在 Windows 上有"窗口边角拖不动 / 拖不到更小尺寸"的问题，已经修。
> 升级到新版安装包后即恢复正常。

### 「窄屏单栏模式」（把 PC 端当通知栏用）

把窗口拖到 **宽度 ≤ 640 像素** 时，前端会自动切换为"移动端单栏模式"：

- 只显示 **会话列表**，右侧聊天面板会被滑到屏幕外；
- 点某条会话，聊天面板才滑进来；点返回又滑出去。

适用场景：把窗口靠在屏幕一边、占一条窄竖条当"消息预览栏"用，工作时一眼能看到谁来消息了。
想恢复 PC 三栏（侧栏 + 列表 + 聊天面板）只需把窗口拖宽到 **641 像素以上** 即可。

## 四、托盘 / 通知 / 截屏

- **托盘图标**：未读消息会在托盘图标上显示红点 / 数字徽章；右键可"显示/隐藏全部窗口"、"快速切账号"、"查看每个账号的未读数"。
- **桌面通知**：新消息会调系统级通知（macOS 通知中心 / Windows 操作中心），可在 **设置 → 关闭桌面通知** 关掉。
- **截屏**：`Ctrl+Shift+A`（macOS：`Command+Shift+A`），框选区域后可直接发送或保存。

## 五、版本更新

软件启动后会自动检查新版本：

- 有新版本时，菜单或设置里会出现红点提示。
- 点 **检查版本 / 更新**，会自动下载并提示你重启完成更新。
- 不想现在更新可以点取消，下次启动还会再提示。

## 六、退出 / 卸载

- **退出某一扇窗**：直接点窗口右上角关闭按钮。窗口会被关掉，但软件本身仍在托盘后台运行，按托盘图标可恢复。
- **真正退出软件**：右键托盘图标 → "退出" / "Quit"。
- **删除某个账号档案**：菜单 **窗口 → 账号管理** → 选中该账号 → 删除。
  ⚠️ 删除会**连同该账号在本地的全部数据 / 缓存一并清掉**，谨慎操作；只是想暂时不用，关窗即可。
- **卸载软件**：
  - Windows：控制面板 → 程序 → 找到 "AI抖商学院" → 卸载。
  - macOS：将 `AI抖商学院.app` 拖到废纸篓。
  - 卸载本身不会清掉本地账号数据；如要彻底清，删掉以下目录：
    - macOS：`~/Library/Application Support/AI抖商学院/`
    - Windows：`%APPDATA%/AI抖商学院/`
    - Linux：`~/.config/AI抖商学院/`

## 七、常见问题速查

| 问题 | 原因 / 解决 |
|---|---|
| 同一个账号在两台电脑登，换了头像但另一台显示旧头像 | v1.0.6 之前的服务端 BUG，已修。**两端都升到最新客户端 + 最新服务端** 即可恢复同步。 |
| Windows 安装包窗口边角拖不动 / 拖不到更小 | v1.0.6 之前的 Electron 配置问题，已修。装新版安装包即可。 |
| 双击图标没反应 | 大概率软件已经在托盘里跑着了，看右下角托盘 / 菜单栏图标，单击呼出即可。 |
| 通知不弹 | 检查系统通知开关；macOS 在"系统设置 → 通知 → AI抖商学院"里允许；Windows 在"设置 → 系统 → 通知"里允许。 |
| 想让不同账号在 Windows 任务栏 **分组显示** | 启动前设环境变量 `TSDD_AUMID=abc.com.tsdaodao.account1`（每个 profile 写一份桌面快捷方式即可）。 |
| 远端域名换了，PC 端连不上 | 这是开发/部署侧的事，看 `apps/web/PACKAGING.md` 的"切换形态"章节，改 `confing.ts` 里 `remoteWebEntryUrl` 后重新打包。 |

## 八、给客服 / 测试的一句话总结

> 不管你看到几扇窗，背后都是同一个客户端进程；每扇窗都能自由缩放；每个账号的数据完全隔离；
> 关掉所有窗后软件并不会真退出（在托盘里）；
> 想真退出请用托盘菜单的"退出"。
